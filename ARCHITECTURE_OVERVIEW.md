## ğŸ¯ Key Components

### Frontend

- **Login Page:** Authenticates users and stores JWT token
- **Dashboard:** Displays real-time merchant statistics
- **API Client:** Manages all backend communication

### Backend

- **Super Admin Controller:** Handles all admin operations
- **Auth Middleware:** Verifies JWT tokens and roles
- **Models:** Define database schema and relationships
- **Routes:** Map URLs to controller functions

### Database

- **super_admins:** Stores admin user credentials
- **activities:** Tracks user actions
- **plans:** Manages subscription plans
- **merchants:** Stores merchant data
