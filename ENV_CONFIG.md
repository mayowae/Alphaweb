# API Configuration for Admin Frontend

Copy this file to `.env.local` and configure your API URL:

```bash
# API Configuration
NEXT_PUBLIC_API_URL=http://localhost:5000

# For production, update this to your backend URL
# NEXT_PUBLIC_API_URL=https://your-backend-domain.com
```

## Setup Instructions

1. Copy this content to a new file named `.env.local` in the root directory
2. Update the `NEXT_PUBLIC_API_URL` to match your backend server URL
3. Restart your Next.js development server for changes to take effect
